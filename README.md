# Experiment-10-RadioButton

Aim

To create an Android application that uses Android Studio RadioButton to select one option from multiple choices.

Algorithm


Start Android Studio.


Create a new Android project.


Select Empty Activity template.


Open activity_main.xml file.


Add a RadioGroup in the layout.


Add multiple RadioButton controls inside the RadioGroup.


Open MainActivity.java file.


Initialize RadioGroup and RadioButton using findViewById().


Set click listener to get selected option.


Run the application and verify the selected option is displayed.
